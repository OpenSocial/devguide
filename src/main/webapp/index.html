OpenSocial The web is better when it's social.

OpenSocial is the open standard for building Applications using only HTML, CSS, and JavaScript that can be run numerous of Open Source and Commercial platforms. OpenSocial incorporates related social web standards like Activity Streams, OAuth, and others.
