OpenSocial Activity Stream

The new inbox

OpenSocial defines JavaScript and web services APIs for accessing user and application specific Activity Streams. OpenSocial 2.0 built on Activity Streams by introducing the ability to embed OpenSocial Applications, and other outside content, into activity entries using something we call 'Embedded Experiences'.
